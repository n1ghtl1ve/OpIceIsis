[How To] How to Install and use Twitter Reporter --------------------------------------------------------------------
1- Install Python 2.7: https://www.python.org/downloads/
2- Install Splinter
- On UNIX, follow these instructions : http://splinter.readthedocs.org/en/latest/install.html.
- On Windows: - Get the zip : https://github.com/cobrateam/splinter/archive/master.zip
- Unzip on your disk, open a terminal (start menu -> type cmd -> launch cmd.exe)
- Go in the folder you unzip splinter (cd XXXX)
- Launch 'python setup.py install'
